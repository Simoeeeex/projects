## Projects

### 1. Gym Tracker
- **Description:** A Python/Tkinter application to log workouts and body measurements.  
- **Features:**
  - Log workouts: exercises, sets, reps, weights.
  - Log body measurements: weight, body fat, chest, arms, legs, waist.
  - Visual charts: weight progress and workout volume.
  - Stats and analysis: total volume, best exercises, weight change.
- **Technologies:** Python, Tkinter, pandas, matplotlib.

---

### 2. Trading Dashboard
- **Description:** Python-based trading app for **crypto and forex markets** with EMA & RSI strategy.  
- **Features:**
  - Real-time price data using `yfinance`.
  - EMA (50/200) and RSI indicators.
  - Automated BUY/SELL/HOLD signals.
  - Paper trading portfolio simulation.
  - Interactive dark-themed Tkinter GUI.
- **Supported Markets:** BTC-USD, ETH-USD, EURUSD=X, GBPUSD=X.
- **Technologies:** Python, Tkinter, matplotlib, yfinance.

---

### 3. Password Strength Analyzer
- **Description:** Python/Tkinter app that checks password strength and generates strong passwords.  
- **Features:**
  - Strength levels: Weak, Medium, Strong, Very Strong.
  - Provides suggestions to improve password security.
  - Generates random strong passwords.
- **Technologies:** Python, Tkinter.

---

### 4. Chess Game (Pygame)
- **Description:** A fully playable chess game using Python and Pygame.  
- **Features:**
  - Complete chessboard setup with pieces.
  - Legal moves for all pieces.
  - Interactive Pygame interface.
- **Limitations:** Check detection and castling do **not** fully work.
- **Technologies:** Python, Pygame.

---

### 5. Trading Journalist
- **Description:** A project related to trading analysis and portfolio management (details based on prior trading strategies).
- **Features:**
  - Data fetching from markets.
  - Indicator calculations (EMA, RSI, etc.).
  - Signal generation and portfolio simulation.
- **Technologies:** Python, pandas, yfinance, matplotlib.
